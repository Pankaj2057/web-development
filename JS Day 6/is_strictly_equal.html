IsStrictlyEqual ( x, y )
The abstract operation IsStrictlyEqual takes arguments x (an ECMAScript language value) and y (an ECMAScript language value) and returns a Boolean. It provides the semantics for the === operator. It performs the following steps when called: 1. If Type(x) is not Type(y), return false. 2. If x is a Number, then a. Return Number::equal(x, y). 3. Return SameValueNonNumber(x, y).
NOTE This algorithm differs from the SameValue Algorithm in its treatment of signed zeroes and NaNs.

The SameValueZero
algorithm behaves similarly to the Strict Equality Comparison (===) in most cases, but with some differences when comparing certain types, especially NaN and -0 (negative zero).
